Welcome to GitHub Pages.

Thank you for visiting our Project Portal. “Student Registration” Java console application created by us will help you create and maintain student data and course offerings at UCI Extension School. Please feel free to download our code files or view at GitHub to test out its functionalities.

Details

You will find two main folders [\resource (containing course & users .txt files) & \srs (containing all .java code files)] for Execution of this program. You can user any open source Java IDE to execute and run these java files.

Functionalities:

  1. View Offered Courses at UCI (including their enrollment capacity & status)
  2. Personalized Student Portal allowing them to Enroll / Drop courses for that semester
  3. Add New Student to Student data file
  4. View Registered Students (First Name and Last Name) at UCI
  5. Also While all the changes made through Console Application it will save changes to both Courses and Users data files
